# SCC85
Seafloor Climate Change based on RCP8.5
- To install the R data package:
```
library(devtools)
# The repository size is quite large
options(timeout=9999999)
install_github("chihlinwei/SCC85")
```
- To access the data layer information:
```
?cimp5_1951_2000_av
?cimp5_2041_2060_av
?cimp5_2081_2100_av
```
